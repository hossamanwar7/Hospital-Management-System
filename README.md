 # Hospital Management System (C++)

A full **Hospital Management System** built using **C++**, applying **Object-Oriented Programming (OOP)** concepts such as inheritance, polymorphism, classes, dynamic memory, and STL structures.

This system manages patients, doctors, prescriptions, queues, and payment operations from the console.
